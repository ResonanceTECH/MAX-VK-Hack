## Ключевая идея решения

Создание современного веб-приложения для образовательной платформы MAX, объединяющего студентов, преподавателей и администрацию в едином цифровом пространстве. Приложение обеспечивает прозрачность коммуникаций, удобный доступ к расписанию, новостям и управлению образовательным процессом.

## Технологический стек

### Frontend

- **React 18.2** с TypeScript 5.3
- **React Router 6** для навигации
- **Vite 5** как сборщик
- **Axios** для HTTP-запросов
- **Lucide React** для базовых иконок
- **CSS Modules** для стилизации

### UI/UX

- Адаптивная верстка (mobile-first подход)
- Единая дизайн-система с CSS переменными
- Современный дизайн с градиентами и анимациями
- Иконки HugeIcons (SVG компоненты)
- Плавные переходы и hover-эффекты
- Интуитивная навигация

### Интеграция

- Интеграция с MAX miniapp
- Авторизация через initData
- Поддержка множественных ролей
- RESTful API для всех операций

## Дизайн-система

### Цветовая палитра

```css
:root {
  /* Основные цвета */
  --primary-50: #f0f9ff;
  --primary-500: #2563eb;
  --primary-600: #1d4ed8;
  
  /* Нейтральные цвета */
  --gray-50: #f8fafc;
  --gray-100: #f1f5f9;
  --gray-200: #e2e8f0;
  --gray-500: #64748b;
  --gray-600: #475569;
  --gray-700: #334155;
  --gray-800: #1e293b;
  
  /* Семантические цвета */
  --success: #10b981;
  --warning: #f59e0b;
  --error: #ef4444;
  --info: #3b82f6;
}
```

### Типографика

- Заголовки: 24-28px, font-weight: 700
- Подзаголовки: 18px, font-weight: 600
- Основной текст: 14-16px, font-weight: 400
- Мета-информация: 12-13px, font-weight: 400

### Эффекты

- Скругления: 12px для карточек, 8px для кнопок
- Тени: `0 4px 12px rgba(0, 0, 0, 0.08)` при hover
- Переходы: `0.3s ease` для всех интерактивных элементов
- Градиенты: для навигации и акцентов

### Компоненты

- **Карточки** - белый фон, скругленные углы, hover-эффекты
- **Кнопки** - синий фон, белый текст, плавные переходы
- **Навигация** - градиентный фон, иконки HugeIcons
- **Формы** - светлый фон, синяя граница при focus
- **Статусы** - цветные бейджи для различных состояний
